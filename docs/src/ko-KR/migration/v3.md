## 지원 범위 변경

- Node.js:

  V3는 최신 LTS Node 버전만 지원합니다.

  현재 LTS 버전은 Node.js 18과 Node.js 20이며, Node.14 및 Node 16에 대한 지원이 제거되었습니다.

- 브라우저:

  V3는 최신 브라우저만 지원하며, Vite의 [기본 브라우저 지원 범위](https://vitejs.dev/guide/build.html#browser-compatibility)와 동일합니다
  - Chrome >=87
  - Firefox >=78
  - Safari >=14
  - Edge >=88

## 기능 제거

- 이제 카운터 요소에 바인딩된 경로를 가져올 때 `id`를 더 이상 읽지 않습니다. `data-path`를 사용해야 합니다.
- 레거시 패키지가 제거되었습니다.

## CSS 변수 업데이트

CSS 변수의 `bgcolor`이 `bg-color`로 변경되었습니다:

```diff
---waline-bgcolor: #fff;
+--waline-bg-color: #fff;
---waline-bgcolor-light: #f8f8f8;
+--waline-bg-color-light: #f8f8f8;
---waline-bgcolor-hover: #f0f0f0;
+--waline-bg-color-hover: #f0f0f0;
---waline-disable-bgcolor: #f8f8f8;
+--waline-disable-bg-color: #f8f8f8;
---waline-code-bgcolor: #282c34;
+--waline-code-bg-color: #282c34;
---waline-info-bgcolor: #f8f8f8;
+--waline-info-bg-color: #f8f8f8;
```
