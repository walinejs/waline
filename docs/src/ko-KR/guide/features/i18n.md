Waline은 내장 다국어 지원을 제공하며, Waline의 로케일 설정을 사용자 정의할 수도 있습니다.

<!-- more -->

## 클라이언트

### 언어 설정

`lang` 옵션을 통해 Waline에서 사용할 언어를 설정할 수 있습니다. 지원되는 언어는 다음과 같습니다:

- zh
- zh-CN
- zh-TW
- en
- en-US
- jp
- jp-JP
- pt-BR
- ru
- ru-RU
- fr-FR
- fr
- vi
- vi-vn
- es
- es-MX

예시:

```js
Waline.init({
  el: '#waline',
  // ...
  lang: 'en',
});
```

::: tip

Waline에 더 많은 언어 지원을 추가하는 데 도움을 주고 싶다면, [client/config/i18n](https://github.com/walinejs/waline/tree/main/packages/client/src/config/i18n)에 PR을 보내주세요.

:::

### 사용자 정의

`locales` 옵션을 통해 언어를 사용자 정의하거나, 일부 필드를 설정하여 기존 UI 텍스트를 덮어쓸 수 있습니다.

자세한 내용은 [쿡북 → 로케일 사용자 정의](../../cookbook/customize/locale.md)를 참조하세요.

## 서버

서버 인터페이스에서 반환되는 정보는 클라이언트가 입력한 lang 매개변수에 따라 언어를 동적으로 전환합니다.

## 사용자 정의

`locales` 옵션을 사용하여 해당 언어의 콘텐츠를 사용자 정의할 수 있습니다. 자세한 내용은 [쿡북 → 로케일 사용자 정의](../../cookbook/customize/locale.md)를 참조하세요.
