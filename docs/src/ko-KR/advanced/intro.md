## 특징

- 빠름
- **진정으로** 안전함
- Markdown 전체 구문 지원
- 간단하고 경량
- 무료로 배포 가능!
- 다양한 배포 및 스토리지 서비스 지원, 최소 243가지 배포 방식 선택 가능!
