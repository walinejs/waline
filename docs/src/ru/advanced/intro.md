::: info Waline

Простая система комментариев с поддержкой серверной части, форк от [Valine](https://valine.js.org).

:::

<!-- more -->

## Возможности

- Быстрая
- **Действительно** Безопасная
- Поддержка полного синтаксиса markdown
- Простая и легкая
- Бесплатное развертывание!
- Поддержка множественных служб развертывания и хранения, как минимум 243 варианта развертывания!

|                                                 | Waline                                                              |                                    |
| ----------------------------------------------- | ------------------------------------------------------------------- | ---------------------------------- |
| **Клиент**                                      | **Сервер**                                                          | **База данных**                    |
