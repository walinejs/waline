Waline には多言語サポートが組み込まれており、ロケール設定をカスタマイズすることもできます。

<!-- more -->

## クライアント

### 言語の設定

`lang` オプションを使って Waline で使用する言語を設定できます。サポートされている言語は以下のとおりです:

- zh
- zh-CN
- zh-TW
- en
- en-US
- jp
- jp-JP
- ko
- ko-KR
- pt-BR
- ru
- ru-RU
- fr-FR
- fr
- vi
- vi-vn
- es
- es-MX

例:

```js
Waline.init({
  el: '#waline',
  // ...
  lang: 'en',
});
```

::: tip

Waline にさらに多くの言語サポートを追加したい場合は、[client/config/i18n](https://github.com/walinejs/waline/tree/main/packages/client/src/config/i18n) への PR を歓迎します。

:::

### カスタマイズ

`locales` オプションを使って言語をカスタマイズしたり、既存の UI テキストの一部を上書きしたりすることができます。

詳細については [クックブック → ロケールのカスタマイズ](../../cookbook/customize/locale.md) を参照してください。

## サーバー

サーバーインターフェースが返す情報は、クライアントから送信される `lang` パラメーターに基づいて動的に言語を切り替えます。

## カスタマイズ

`locales` オプションで対応する言語のコンテンツをカスタマイズできます。詳細は [クックブック → ロケールのカスタマイズ](../../cookbook/customize/locale.md) を参照してください。
