## サポート範囲の変更

- Node.js:

  V3 は最新の LTS Node バージョンのみをサポートします。

  現在の LTS バージョンは Node.js 18 と Node.js 20 であり、Node.14 と Node 16 のサポートは削除されました。

- ブラウザ:

  V3 はモダンブラウザのみをサポートし、Vite の [デフォルトブラウザサポート範囲](https://vitejs.dev/guide/build.html#browser-compatibility) に準拠します。
  - Chrome >=87
  - Firefox >=78
  - Safari >=14
  - Edge >=88

## 機能の削除

- カウンター要素にバインドされたパスを取得する際、`id` は読み取られなくなりました。`data-path` を使用してください。
- レガシーパッケージが削除されました。

## CSS 変数の更新

CSS 変数内の `bgcolor` が `bg-color` に変更されました：

```diff
---waline-bgcolor: #fff;
+--waline-bg-color: #fff;
---waline-bgcolor-light: #f8f8f8;
+--waline-bg-color-light: #f8f8f8;
---waline-bgcolor-hover: #f0f0f0;
+--waline-bg-color-hover: #f0f0f0;
---waline-disable-bgcolor: #f8f8f8;
+--waline-disable-bg-color: #f8f8f8;
---waline-code-bgcolor: #282c34;
+--waline-code-bg-color: #282c34;
---waline-info-bgcolor: #f8f8f8;
+--waline-info-bg-color: #f8f8f8;
```
