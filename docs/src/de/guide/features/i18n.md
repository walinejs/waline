Waline verfügt über integrierte mehrsprachige Unterstützung, und Sie können auch die Locale-Konfiguration von Waline anpassen.

<!-- more -->

## Client

### Sprache festlegen

Sie können die von Waline verwendete Sprache über die Option `lang` festlegen. Die unterstützten Sprachen sind:

- zh
- zh-CN
- zh-TW
- en
- en-US
- jp
- jp-JP
- ko
- ko-KR
- pt-BR
- ru
- ru-RU
- fr-FR
- fr
- vi
- vi-vn
- es
- es-MX
- de (Deutsch)

Beispiel:

```js
Waline.init({
  el: '#waline',
  // ...
  lang: 'de',
});
```

::: tip

Wenn Sie Waline helfen möchten, weitere Sprachunterstützung hinzuzufügen, sind wir dankbar, wenn Sie einen PR an [client/config/i18n](https://github.com/walinejs/waline/tree/main/packages/client/src/config/i18n) senden.

:::

### Anpassen

Sie können die Sprache über die Option `locales` anpassen oder einige Felder festlegen, um den vorhandenen UI-Text zu überschreiben.

Für Details siehe [Cookbook → Locale anpassen](../../cookbook/customize/locale.md).

## Server

Die vom Server-Interface zurückgegebenen Informationen wechseln dynamisch die Sprache basierend auf dem vom Client eingegebenen lang-Parameter.

## Anpassen

Sie können den Inhalt für die entsprechende Sprache mit der Option `locales` anpassen. Siehe [Cookbook → Locale anpassen](../../cookbook/customize/locale.md) für Details.
