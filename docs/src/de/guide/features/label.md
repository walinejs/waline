Waline ermöglicht es Ihnen, Level-Label basierend auf Interaktionen und benutzerdefinierte Label für angemeldete Benutzer festzulegen.

## Grad-Label

Sie müssen die Variable `LEVELS` auf dem Server konfigurieren, um diese Funktion zu aktivieren. Siehe [Server-Umgebungsvariablen-Konfiguration](../../reference/server/env.md#display).

Sie können diese Level-Label über die Locales-Option anpassen. Siehe [Mehrsprachig](./i18n.md#customize)

## Exklusives Label

Sie können Label für Benutzer über die Verwaltungsseite anpassen.

![Admin Label](./assets/label-admin.jpg)

Als angemeldeter Benutzer können Sie Ihr exklusives Label auch auf der Profilseite aktualisieren.

![Profile Label](./assets/label-profile.jpg)
