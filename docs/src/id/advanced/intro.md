::: info Waline

Sistem komentar sederhana dengan dukungan backend yang di-fork dari [Valine](https://valine.js.org).

:::

<!-- more -->

## Fitur

- Cepat
- **Benar-benar** Aman
- Dukungan sintaks markdown lengkap
- Sederhana dan ringan
- Deploy gratis!
- Dukungan berbagai metode deploy dan layanan penyimpanan, setidaknya 243 pilihan gaya deploy!

|                                                 | Waline                                                              |                                    |
| ----------------------------------------------- | ------------------------------------------------------------------- | ---------------------------------- |
| **Client**                                      | **Server**                                                          | **Database**                       |
| [@waline/client](https://waline.js.org)         | [Vercel](https://vercel.com)                                        | [MongoDB](https://mongodb.com)     |
