Waline memiliki dukungan multibahasa bawaan, dan Anda juga dapat mengkustomisasi konfigurasi lokal Waline.

<!-- more -->

## Klien

### Mengatur Bahasa

Anda dapat mengatur bahasa yang digunakan oleh Waline melalui opsi `lang`. Bahasa yang didukung adalah:

- zh
- zh-CN
- zh-TW
- en
- en-US
- jp
- jp-JP
- ko
- ko-KR
- pt-BR
- ru
- ru-RU
- fr-FR
- fr
- vi
- vi-vn
- es
- es-MX

Contoh:

```js
Waline.init({
  el: '#waline',
  // ...
  lang: 'en',
});
```

::: tip

Jika Anda ingin membantu Waline menambahkan dukungan bahasa lebih banyak, kami mengundang Anda untuk mengirimkan PR ke [client/config/i18n](https://github.com/walinejs/waline/tree/main/packages/client/src/config/i18n).

:::

### Kustomisasi

Anda dapat mengkustomisasi bahasa melalui opsi `locales`, atau menetapkan beberapa bidang untuk menimpa teks UI yang sudah ada.

Untuk detailnya, lihat [Cookbook → Kustomisasi Lokal](../../cookbook/customize/locale.md).

## Server

Informasi yang dikembalikan oleh antarmuka server akan secara dinamis beralih bahasa berdasarkan parameter lang yang dikirimkan oleh klien.

## Kustomisasi

Anda dapat mengkustomisasi konten untuk bahasa yang sesuai dengan opsi `locales`. Lihat [Cookbook → Kustomisasi Lokal](../../cookbook/customize/locale.md) untuk detailnya.
