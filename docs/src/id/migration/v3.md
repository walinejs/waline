## Perubahan Dukungan

- Node.js:

  V3 hanya mendukung versi LTS Node terbaru.

  Versi LTS saat ini adalah Node.js 18 dan Node.js 20, yang berarti kami telah menghapus dukungan untuk Node.14 dan Node 16.

- Browser:

  V3 hanya mendukung browser modern, selaras dengan [Rentang Dukungan Browser Default](https://vitejs.dev/guide/build.html#browser-compatibility) Vite
  - Chrome >=87
  - Firefox >=78
  - Safari >=14
  - Edge >=88

## Penghapusan Fitur

- Sekarang, saat mendapatkan path yang terikat ke elemen counter, `id` tidak lagi dibaca. Anda harus menggunakan `data-path`.
- Paket-paket lama telah dihapus.

## Pembaruan Variabel CSS

`bgcolor` dalam variabel CSS diubah menjadi `bg-color`:

```diff
---waline-bgcolor: #fff;
+--waline-bg-color: #fff;
---waline-bgcolor-light: #f8f8f8;
+--waline-bg-color-light: #f8f8f8;
---waline-bgcolor-hover: #f0f0f0;
+--waline-bg-color-hover: #f0f0f0;
---waline-disable-bgcolor: #f8f8f8;
+--waline-disable-bg-color: #f8f8f8;
---waline-code-bgcolor: #282c34;
+--waline-code-bg-color: #282c34;
---waline-info-bgcolor: #f8f8f8;
+--waline-info-bg-color: #f8f8f8;
```
