::: info Waline

Un semplice sistema di commenti con supporto backend biforcato da [Valine](https://valine.js.org).

:::

<!-- more -->

## Caratteristiche

- Veloce
- **Veramente** Sicuro
- Supporto per la sintassi markdown completa
- Semplice e leggero
- Distribuzione gratuita!
- Supporto per più servizi di distribuzione e archiviazione, almeno 243 scelte di stile di distribuzione!
